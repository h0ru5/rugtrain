About Rugbears Training Site

Purpose

This training site is used to coordinate the trainings and other events of the Munich Rugbears, a wheelchair-rugby team based in Munich, Germany

It is also used as my playground for experiments with contemporary web design tolls, libraries and practices

Technology

The original app was an ugly hacked thing using PHP with mixed presentation and code.

It was reworked for flexibility and maintainability, in the rework it was (mostly) seperated into a MVC-architecture
The Model is a plain MySQL db using MyISAM, there is no abstraction layer or ORM (yet). Database access is only abstracted via PEAR::MDB2
These databadse calls are abstracted into a RESTful controller tier using the exceptionally well-designed epiphany framework by jmathai

There are currently 4 different views that access this layer:

/retro
A backport of the old design with mixed logic and presentation, no overview but only a viewfor a single event.
